Brief

For AutoTechnik, I built a custom WordPress website using Laravel Blade, ES6 JavaScript, and Webpack.

This project was unorthodox compared to previous builds I have done, since the senior designer wanted to use Beaver Builder heavily. As a result, I had to be meticulous with how I built my custom modules, and extended them to the page-builder so that it was easy for the designer and client to add or edit content while still maintaining consistency with the custom code.

Since it became necessary for both myself and the senior designer to work on the website concurrently, I made heavy use of shell scripts to be able to work seamlessly between the local and staging environments, including syncing the databases and deploying within seconds. Automation also meant that I could work with the designer in real-time to make style and functionality changes while sitting together, which boosted our productivity greatly as the design was evolving during the build phase.

Writing the CSS in a modular way was also crucial in ensuring that style changes could be made swiftly, without entangling the existing style rules.
